# GET STARTED

Symfony 4 with basic vendors

**Twig**,
**Maker**,
**Profiler**,
**Annotations**,
**Asset**,
**Translation**,
**Encore with Sass**


# Installation

1. Clone the project in your computer:

``git clone https://github.com/Zhartas/interface.git``


2. Install Symfony vendors

``composer install``

3. Install 

``yarn install``

4. ENJOY 

``php bin/console server:run``
